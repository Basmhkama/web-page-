1. บทนำ
ในปัจจุบัน ปัญหาขยะพลาสติกถือเป็นหนึ่งในปัญหาสิ่งแวดล้อมที่ส่งผลกระทบอย่างรุนแรงต่อโลก ทั้งในด้านสิ่งแวดล้อม ระบบนิเวศ และสุขภาพของมนุษย์ ขยะพลาสติกจำนวนมากไม่สามารถย่อยสลายได้ตามธรรมชาติ และสะสมอยู่ในแหล่งน้ำ ป่าไม้ และทะเล
สื่อโมชั่นกราฟิกเป็นสื่อดิจิทัลที่สามารถถ่ายทอดข้อมูลที่ซับซ้อนให้เข้าใจได้ง่าย ผ่านภาพเคลื่อนไหวและการเล่าเรื่องที่น่าสนใจ โครงงานนี้จึงจัดทำขึ้นเพื่อใช้โมชั่นกราฟิกเป็นสื่อในการสร้างความตระหนักรู้เกี่ยวกับปัญหาขยะพลาสติก และนำเสนอผ่านเว็บไซต์เพื่อให้ผู้ชมสามารถเข้าถึงผลงานได้สะดวกมากยิ่งขึ้น

2. แนวคิดและที่มาของโครงงาน
แนวคิดของโครงงานนี้เกิดจากการตระหนักถึงผลกระทบที่เกิดจากการใช้พลาสติกในชีวิตประจำวันของมนุษย์ ซึ่งส่งผลเสียต่อสิ่งแวดล้อมในระยะยาว ผู้จัดทำจึงต้องการนำเสนอเรื่องราวของขยะพลาสติกในรูปแบบที่เข้าใจง่ายและสร้างความรู้สึกร่วมแก่ผู้ชม
โครงงานได้นำตัวละครหุ่นยนต์มาเป็นผู้เล่าเรื่อง เพื่อถ่ายทอดสิ่งที่พบเห็นจากผลกระทบของขยะพลาสติก เช่น การทำลายธรรมชาติและสัตว์ต่าง ๆ การเล่าเรื่องผ่านตัวละครสมมติช่วยให้ผู้ชมสามารถเข้าถึงเนื้อหาได้ง่ายและเกิดความสนใจมากยิ่งขึ้น

3. วัตถุประสงค์ของโครงงาน
วัตถุประสงค์ของโครงงานนี้ ได้แก่
เพื่อสร้างความตระหนักรู้เกี่ยวกับปัญหาและอันตรายของขยะพลาสติก
เพื่อใช้โมชั่นกราฟิกเป็นสื่อในการสื่อสารด้านสิ่งแวดล้อม
เพื่อพัฒนาทักษะด้านการออกแบบโมชั่นกราฟิกแบบสองมิติ
เพื่อฝึกการนำผลงานโมชั่นกราฟิกไปแสดงผลผ่านเว็บไซต์

4. กลุ่มเป้าหมาย
กลุ่มเป้าหมายของโครงงานนี้ ได้แก่ นักเรียน นักศึกษา และบุคคลทั่วไปที่สนใจปัญหาสิ่งแวดล้อม โดยมีช่วงอายุประมาณ 15–30 ปี ซึ่งเป็นกลุ่มที่สามารถเข้าถึงสื่อดิจิทัลและอินเทอร์เน็ตได้ง่าย

5. ประเภทและลักษณะของโมชั่นกราฟิก
โครงงานนี้เป็นโมชั่นกราฟิกแบบสองมิติ (2D) มีความยาวประมาณ 36 วินาที รูปแบบการนำเสนอเป็นข้อความเคลื่อนไหวร่วมกับภาพประกอบ โดยไม่มีเสียงบรรยายหรือดนตรีประกอบ เน้นการสื่อสารผ่านภาพและข้อความอย่างเรียบง่าย

6. โครงเรื่องและลำดับเหตุการณ์
เนื้อเรื่องเริ่มต้นด้วยการแนะนำตัวละครหุ่นยนต์ จากนั้นหุ่นยนต์จะเล่าเรื่องเกี่ยวกับผลกระทบของขยะพลาสติกที่มีต่อสิ่งแวดล้อม เช่น การปนเปื้อนในธรรมชาติ การทำร้ายสัตว์บกและสัตว์ทะเล
ในช่วงท้ายของเรื่อง หุ่นยนต์จะส่งสารถึงมนุษย์เพื่อกระตุ้นให้เกิดความตระหนักรู้และร่วมกันดูแลรักษาสิ่งแวดล้อม เนื่องจากโลกเป็นสิ่งที่ทุกคนต้องช่วยกันรับผิดชอบ



7. เครื่องมือและเทคโนโลยีที่ใช้
เครื่องมือและเทคโนโลยีที่ใช้ในการจัดทำโครงงาน ได้แก่
Canva สำหรับออกแบบโมชั่นกราฟิก
PhotoRoom สำหรับปรับแต่งภาพ
Visual Studio Code สำหรับพัฒนาเว็บไซต์
HTML, CSS และ JavaScript สำหรับการเขียนโค้ดเว็บไซต์

8. การออกแบบและพัฒนาเว็บแอปพลิเคชัน
โครงงานนี้ได้พัฒนาเว็บไซต์แบบหน้าเดียว (Single Page Website) เพื่อใช้แสดงวิดีโอโมชั่นกราฟิก โดยไม่มีการใช้ฐานข้อมูลหรือระบบหลังบ้าน การออกแบบเว็บไซต์เน้นความเรียบง่าย ใช้งานสะดวก และเหมาะสมกับผู้ใช้งานทั่วไป

9. ผลลัพธ์ที่ได้รับจากโครงงาน
จากการจัดทำโครงงานนี้ ได้ผลงานโมชั่นกราฟิกที่สามารถสื่อสารประเด็นด้านสิ่งแวดล้อมได้อย่างชัดเจน ผู้จัดทำได้รับประสบการณ์ในการออกแบบสื่อดิจิทัลและการพัฒนาเว็บไซต์ รวมถึงได้พัฒนาทักษะด้านเทคนิคและความคิดสร้างสรรค์

10. บทสรุป
โครงงานการเดินทางของขยะพลาสติกแสดงให้เห็นถึงความสำคัญของการใช้โมชั่นกราฟิกเป็นสื่อในการสร้างความตระหนักรู้ด้านสิ่งแวดล้อม ผ่านการเล่าเรื่องที่เข้าใจง่ายและเข้าถึงผู้ชมได้หลากหลายกลุ่ม โครงงานนี้จึงเป็นประสบการณ์การเรียนรู้ที่มีคุณค่าและเป็นประโยชน์ต่อการศึกษาและการพัฒนาทักษะของผู้จัดทำ
